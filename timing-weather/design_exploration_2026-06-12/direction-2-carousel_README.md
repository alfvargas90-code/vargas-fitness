# Direction 2 — System Carousel

**File:** `direction-2-carousel.html`

## The user moment it serves
The **explorer.** When Alfredo wants to sit *inside* one system at a time instead of
scanning all four at once. Each lens — Vedic · Modern · Traditional · BaZi — gets the full
screen as a hero card with its own accent color, and a horizontal swipe moves between them.
It feels like consulting four trusted advisors one after another.

## Why it fits Alfredo specifically
- He's a **parallel tab-thinker** — but tabs are sequential, not simultaneous. The carousel
  honors that: full attention on one frame, deliberate swipe to the next.
- Each system keeps equal weight (no auto-pick), which respects that he values all four
  traditions and the **cross-system convergence** is itself the insight.
- Color-coding each lens (Vedic→pink, Modern→cyan, Traditional→violet, BaZi→coral) builds a
  fast visual vocabulary over time.

## Interaction
Touch-swipe or tap the dot labels at the bottom. The moon strip + header stay pinned; only
the focal card changes. Snap transition, 0.42s ease.

## Aesthetic notes
Same deep blue/purple world, compact orbiting-moon header strip (saves vertical space for
the hero card), per-system tinted glass gradients, pink/coral italic prose, "lean in" gold
footer line per system, animated dot-bar indicator.

## Chart data surfaced
Each card carries that system's live read: Vedic Jupiter-return + Venus antardasha · Modern
solar-arc Jupiter→Sun + 12th profection · Traditional Jupiter-year-from-the-12th + Moon void
· BaZi Bing-Wu peer-fire + Rabbit operating self + Jupiter-return stack.

## Tradeoff
Requires interaction to see everything — three of four systems are always one swipe away,
not visible. Worse for a fast glance, better for a daily deep-read.
